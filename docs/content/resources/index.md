## Using These Resources

### For Individuals

If you're learning about the model or considering starting a cell:

1. Start with **[The Model](../model/)** to understand the framework
2. Review **[Security Practices](../security/)** to understand operational security
3. Explore **[Technical Tools](./technical)** to see what platforms you'll use
4. Download **[Rights Cards](../rights/print-cards)** to carry with you

### For Founding Teams

If you're starting a cell with 3-5 trusted people:

1. Complete **[Founding Team Guide](../start-a-cell/founding-team)**
2. Set up **[Signal](../start-a-cell/signal-setup)** for all founders
3. Use **[Printable Checklists](./printable)** to track progress
4. Review **[Onboarding Process](../start-a-cell/onboarding)** before adding members

### For Active Cells

If you're operating an established cell:

1. Conduct monthly **[Security Audits](./printable#security-audit-checklist)**
2. Use **[Templates](./printable)** for consistent processes
3. Distribute **[Rights Cards](../rights/print-cards)** to community members
4. Review **[Security Practices](../security/)** regularly with all members
