## Liisaska Hubinta

### Liiska Hubinta Kooxda Aasaasiga

Tan isticmaal marka aad bilowday unug cusub 3-5 qof oo la isku halayn karo.

```
LIISKA HUBINTA BILOWGA KOOXDA AASAASIGA

Ka Hor Bilowga:
[ ] 3-5 aasaasayaal la aqoonsaday oo ballan qaatay
[ ] Dhammaan aasaasayaashu si qoto dheer ayay isku halayaan
[ ] Mabaadi'da Aasaasiga la dib u eegay oo lagu heshiiyay
[ ] Tababarka amniga la dhamaystir wada
[ ] Kaabayaasha Signal la dejiyay
[ ] Doorarka maamulayaasha la xilsaaray
[ ] Habka go'aan qaadashada lagu heshiiyay
[ ] Jadwalka kulammada la dhisay

Bisha Kowaad:
[ ] Qaybaha bilowga ah la qeexay
[ ] Habka soo dhaweynta la dukumeentiyay
[ ] Xubnaha kowaad la aqoonsaday amaanka
[ ] Hawlaha kowaad la qorsheeyay
[ ] Hab-maamuulka degdegga ah la dhisay
```

---

### Liiska Hubinta Soo Dhaweynta Xubnaha Cusub

Tan isticmaal xubin kasta oo cusub oo ku biiraya unugaaga.

```
LIISKA HUBINTA SOO DHAWEYNTA

Amaanka:
[ ] Amaanku xaqiijiyaa inuu ku kalsoon yahay musharaxa
[ ] Maamulaha Soo Dhaweynta la ogeysiiyay
[ ] Wadahadalka bilowga ah la jadwalay

Wadahadalka Bilowga ah:
[ ] Maamulaha, amaanka, iyo musharaxu way kulmaan
[ ] Unugga la sharxay, su'aalaha la jawaabay
[ ] Ballan qaadashada iyo filashada cad
[ ] Go'aanka la gaadhay in la sii wado (ama maya)

Tababarka Amniga:
[ ] Signal la rakibay oo la dejiyay
[ ] Xirka shaashadda la daaray
[ ] Xirka diiwaan gelinta la daaray
[ ] Farriimaha libdhaya la sharxay
[ ] Amniga qalabka la dib u eegay
[ ] Qaybsanaanta la sharxay
[ ] Waxa AANAN la wadaagin la dib u eegay
[ ] Hab-maamuulka khatar-galka la dib u eegay

Bilowga Mudada Tijaabada:
[ ] Lagu daray koox qaybeed
[ ] Amaanka iyo hogaamiyaha qaybta la ogeysiiyay
[ ] Hawlaha kowaad la xilsaaray
[ ] Jadwalka hubinta la dejiyay

Mudada Tijaabada Dhamaystiran:
[ ] Qiimaynta hogaamiyaha qaybta
[ ] Jawaab celinta amaanka
[ ] Xaaladda kalsoonida la xaqiijiyay
[ ] Helitaanka buuxa ee qaybta la siiyay
```

---

### Liiska Hubinta Hubinta Amniga

Samee hubintan bishii dhammaan xubnaha.

```
HUBINTA AMNIGA BISHII

Dhammaan Xubnaha:
[ ] Signal weli si sax ah ayaa loo dejiyay
[ ] Xirka telefoonka la daaray
[ ] Xirka diiwaan gelinta la daaray
[ ] Farriimaha libdhaya firfircoon
[ ] Macluumaad xasaasi ah ma jiraan sawirrada shaashadda
[ ] Faahfaahinta hawlgalka ma jiraan apps kale

Hogaamiyayaasha Qaybta:
[ ] Dhammaan xubnahu waxay raacayaan qaybsanaanta
[ ] Ma jiro wadaagga macluumaadka qaybaha dhexdooda
[ ] Wadooyinka isgaarsiintu waa kuwo ammaan ah
[ ] Ma jiraan walaacyo cusub oo amni

Maamulayaasha:
[ ] Habka amaanka la raacayo
[ ] Ma jiraan walaacyo kalsooneed oo ku saabsan xubnaha
[ ] Hab-maamuulka degdegga ah weli hadda
[ ] Isgaarsiinta kaabada la tijaabiyay
```

## Tilmaamaha Daabacaadda

### Kaardka Xuquuqda
- Daabac warqad adag si ay u adkayso
- Jar xariiqaha dhibcaha ah
- Laab cabbirka jeebka
- Laminate haddii suurtogal ah

### Liisaska Hubinta
- Daabac warqad caadi ah
- Isticmaal mid xubin/hawl kasta
- Burburiso ka dib dhamaadka
- Weligaa ha kaydsan dijitaal ahaan

### Kaardka Doorka
- Daabac warqad adag
- Sii xubnaha inta lagu jiro soo dhaweynta
- Bedel haddii la lumo (ha haysan nuqulo)

---

::: warning XUSUUSINTA AMNIGA
**Weligaa ha kaydsan foomamka la buuxiyay dijitaal ahaan.** Hayo nuqulo jireed meel ammaan ah. Burburiso foomamka marka xubnahu ka baxaan unugga.
:::
