# احصل على الدعم

<div dir="rtl">

إذا كنت أنت أو شخص تعرفه بحاجة إلى المساعدة، فأنت لست وحدك. شبكة مجتمعنا هنا لمساعدتك.

::: warning الخصوصية مهمة للغاية
نجمع فقط الحد الأدنى من المعلومات اللازمة لمساعدتك. ليس لدينا سجلات للأشخاص الذين نساعدهم. أمنك هو أولويتنا.
:::

## أنواع الدعم المتاحة

اعتمادًا على قدرة خليتك المحلية، قد يشمل الدعم:

| نوع الدعم | الوصف |
|--------------|-------------|
| **النقل** | رحلات آمنة إلى العمل، المواعيد، متاجر البقالة، تواريخ المحكمة |
| **اعرف حقوقك** | تدريب على حقوقك القانونية أثناء اللقاءات |
| **الاتصال القانوني** | إحالات إلى محامي الهجرة والمساعدة القانونية |
| **الموارد الطارئة** | الطعام، الإمدادات، والمساعدة النقدية الطارئة |
| **دعم الأسرة** | مساعدة رعاية الأطفال، تنسيق المدرسة إذا تم احتجاز أحد أفراد الأسرة |

## كيفية طلب الدعم

### إذا كنت تعرف شخصًا في الشبكة

الطريقة الأكثر أمانًا للاتصال هي من خلال شخص تعرفه وتثق به بالفعل وهو جزء من خلية محلية. اطلب منهم ربطك بالدعم المناسب.

### إذا كنت لا تعرف أحدًا

استخدم النموذج أدناه للاتصال. سيتواصل معك عضو مجتمعي عبر قناة آمنة.

## ما التالي

1. سيتواصل معك عضو مجتمعي عبر طريقتك المفضلة
2. سيؤكدون احتياجاتك ويربطونك بالدعم الصحيح
3. ستكون جميع الاتصالات عبر قنوات آمنة
4. لن يُطلب منك أبدًا وثائق أو إثبات الحالة

## حالات الطوارئ

**إذا كنت أنت أو شخص ما يتم احتجازه الآن:**
- ابق هادئًا
