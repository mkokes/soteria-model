# نموذج سوتيريا

نموذج سوتيريا هو إطار عمل للمساعدة المتبادلة والحماية المنظمة من قبل المجتمع. تم تصميمه ليكون **آمناً** و**قابلاً للتوسع** و**قابلاً للتكرار** — يمكن لأي مجتمع تبنيه.

## نظرة عامة على البنية

```
                    ┌─────────────────────────────┐
                    │   SOTERIAMODEL.COM          │
                    │   (النموذج العام)           │
                    └──────────────┬──────────────┘
                                   │
           ┌───────────────────────┼───────────────────────┐
           │                       │                       │
    ┌──────▼──────┐         ┌──────▼──────┐         ┌──────▼──────┐
    │  المدينة أ  │         │  المدينة ب  │         │  مدينتك     │
    │  الخلية     │         │  الخلية     │         │  الخلية     │
    └──────┬──────┘         └──────┬──────┘         └──────┬──────┘
           │                       │                       │
     ┌─────┴─────┐           ┌─────┴─────┐           ┌─────┴─────┐
     │ الأقسام   │           │ الأقسام   │           │ الأقسام   │
     │ المشرفون  │           │ المشرفون  │           │ المشرفون  │
     └───────────┘           └───────────┘           └───────────┘
```

كل **خلية** هي شبكة محلية مستقلة. داخل كل خلية توجد **أقسام** — فرق متخصصة بأدوار مختلفة. **المشرفون** ينسقون الأنشطة دون تجاوز حدود الأمان.

## المكونات الرئيسية

| المكون | الغرض |
|--------|-------|
| [**الخلايا**](/locales/ar/model/cells) | شبكات محلية مستقلة، مبنية على الجغرافيا أو المجتمع |
| [**الأقسام**](/locales/ar/model/segments) | فرق وظيفية (الحراس، الناقلون، الموارد، إلخ) |
| [**المشرفون**](/locales/ar/model/stewards) | أدوار التنسيق (قادة الأقسام، اللوجستيات، الاستقبال) |
| [**مستويات الثقة**](/locales/ar/model/trust) | نظام التزكية والتقدم من مرشح إلى عضو موثوق |

## نموذج الأمان

نموذج سوتيريا مبني على **التجزئة**:

- الأقسام معزولة عن بعضها البعض
- عضو في قسم لا يعرف أعضاء قسم آخر
- إذا تم اختراق شخص، يتم احتواء الضرر
- المنسقون يعرفون منسقين آخرين، ليس كل الأعضاء

هذا ليس جنون الارتياب — إنه حماية. في البيئات عالية المخاطر، ما لا يعرفه الناس، لا يمكنهم الكشف عنه.

## التكنولوجيا

كل التنسيق يحدث عبر **اتصالات مشفرة خارج المنصة**:

- **Signal** للرسائل الجماعية (مجموعات منفصلة لكل قسم)
- **لا توجد قواعد بيانات مركزية** لأعضاء المجتمع أو الأشخاص المعرضين للخطر
- **هذا الموقع عام** — يحتوي فقط على النموذج، أبداً التفاصيل التشغيلية
