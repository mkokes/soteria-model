# Estructura de Células

Una **célula** es la unidad organizacional fundamental del Modelo Soteria. Es una red local y autónoma que opera independientemente.

## ¿Qué es una Célula?

Una célula es una red de ayuda mutua basada en la comunidad, típicamente organizada alrededor de:

- Una **ciudad o pueblo**
- Un **vecindario** en áreas metropolitanas grandes
- Una **región geográfica** en áreas rurales
- Una **comunidad** (cultural, religiosa, laboral)

│                  │(Coordinadores)│                 │

Cada célula contiene:
- **Segmentos** — Equipos funcionales (ver [Segmentos](/locales/es/model/segments))
- **Guías** — Roles de coordinación (ver [Guías](/locales/es/model/stewards))
- **Miembros** — Individuos de confianza dentro de los segmentos

## Independencia de las Células

Las células operan autónomamente:

| Lo que las Células Comparten | Lo que las Células NO Comparten |
|------------------------------|--------------------------------|
| Este modelo | Listas de miembros |
| Actualizaciones legales | Detalles operativos |
| Avisos de seguridad | Comunicaciones internas |
| Mejoras de plantillas | Ubicaciones o rutas |

No hay "organización nacional Soteria." Cada célula es soberana.

## Iniciando una Célula

Una célula comienza con un **equipo fundador** de 3-5 individuos de confianza que:

1. Se conocen bien y confían profundamente entre sí
2. Se comprometen con los [Principios Fundamentales](/locales/es/model/principles)
3. Completan entrenamiento de seguridad juntos
4. Establecen segmentos iniciales basados en necesidades locales
5. Comienzan a avalar nuevos miembros cuidadosamente

Ver [Iniciar una Célula](/locales/es/start-a-cell/) para orientación detallada.

## Escalando una Célula

A medida que las células crecen, pueden elegir subdividirse. Esto es **opcional** — cada célula decide qué funciona para su contexto.

| Situación | Opciones |
|-----------|---------|
| Segmento creciendo grande (15+ miembros) | Agregar co-líderes, dividir por función o geografía |
| Célula cubriendo área amplia | Crear sub-células regionales o coordinadores de área |
| Comunidad pequeña/remota | Mantener célula única indefinidamente |

La **guía de 15 personas**: Grupos mayores de ~15 se vuelven más difíciles de mantener confianza y coordinar. Cuando los segmentos crecen, considera la subdivisión — pero siempre es una decisión local.

## Seguridad de la Célula

- Ninguna célula debe tener conocimiento completo de otra célula
- Si las células coordinan, es solo a través de enlaces designados
- Una célula comprometida no puede comprometer la red
