# Qaababka iyo Qalabka

Qaabab la habeyn karo si aad u bilowdo unugga.

## Qaabka Heshiiska Kooxda Aasaasiga ah

```markdown
# Heshiiska Kooxda Aasaasiga ah

Annaga, kuwa hoosta saxiixa, waxaan ku ballan qaadnay inaan aasaasno unug Soteria si aan u ilaalino oo aan u taageerro [bulshada/kooxda].

## Ballanqaadyadayada

Waxaan ku ballan qaadnay:
- [ ] Mabaadi'da aasaasiga ah ee qaabka Soteria
- [ ] Ilaalinta dhaqannada amniga adag
- [ ] Qaybinta, xitaa dhexdayada
- [ ] Ballanqaadka ugu yaraan hal sano
- [ ] Taageeridda midba midka kale shaqadan

## Qiyamkayaga La wadaago

[Qor qiyamka gaarka ah ee kooxdaada]

## Sida aan u Qaadanno Go'aamada

[Heshiis / Aqlabiyadda / Kale — caddee habkaaga]

## Sida aan ula Tacaalino Khilaafaadka

[Habkaaga khilaafaadka]

## Saxiixyada

- [Magaca] — [Taariikhda]
- [Magaca] — [Taariikhda]
- [Magaca] — [Taariikhda]
```

## Liiska Hubinta Soo Gelitaanka

```markdown
# Liiska Hubinta Xubin Cusub

**Magaca Musharaxa:** _______________
**Taageeruhu:** _______________
**Taariikhda La soo jeediyay:** _______________

## Marxaladda 1: Taageero
- [ ] Taageeruhu wuxuu yaqaan musharaxa 6+ bilood
- [ ] Taageeruhu wuxuu ku kalsoon yahay musharaxa macluumaad xasaasi ah
- [ ] Taageeruhu wuxuu markhaati ka yahay qiyamka musharaxa iyo ballanqaadka
- [ ] Taageeruhu wuxuu fahmaa mas'uuliyadda

## Marxaladda 2: Kulanka Bilowga ah
- [ ] Taageeruhu wuxuu sharaxay ujeedada unugga
- [ ] Taageeruhu wuxuu sharaxay filashada ballanqaadka
- [ ] Taageeruhu wuxuu sharaxay shuruudaha amniga
- [ ] Musharaxu wuxuu muujiyay xiisaha sii wadista

## Marxaladda 3: Oggolaanshaha Maamulayaasha
- [ ] Taageeruhu wuxuu u soo bandhigay musharaxa maamulayaasha
- [ ] Maamulayaashu way dib u eegeen taageerida
- [ ] Maamulayaashu way oggolaadeen sii wadista
- [ ] **Taariikhda:** _______________

## Marxaladda 4: Tababarka Amniga
- [ ] Musharaxu wuxuu akhriyay buugga amniga
- [ ] Musharaxu wuxuu rakibay oo habeeyay Signal
- [ ] Musharaxu wuxuu ka soo qayb galay fadhiga tababarka amniga
- [ ] Tababaruhu wuxuu xaqiijiyay fahamka
- [ ] **Tababaruhu:** _______________ **Taariikhda:** _______________

## Marxaladda 5: Qoondeynta Qaybta
- [ ] Qaybta loo qoondeeyay: _______________
- [ ] Hogaamiyaha qaybta waa loo sheegay
- [ ] Hogaamiyaha qaybta wuu oggoladay

## Marxaladda 6: Ku Darista Signal
- [ ] Lagu daray kooxda qaybta
- [ ] Xaqiijiyay lambarka amniga hogaamiyaha qaybta
- [ ] Xaqiijiyay lambarka amniga taageeruha
- [ ] Lagu daray kooxda ogeysiisyada (haddii jirto)

## Marxaladda 7: Isku Dhafka
- [ ] Taageeruhu wuxuu hubiyay kadib 1 toddobaad
- [ ] Hogaamiyaha qaybta wuxuu hubiyay kadib 1 toddobaad
- [ ] Dib u eegista hal bil la dhammaystay
- [ ] **Xaalada:** _______________
```
