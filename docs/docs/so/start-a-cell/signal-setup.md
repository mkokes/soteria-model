# Dejinta Signal

Signal waa madal isgaarsiinta ee lagu talinayo unugyada Soteria. Waxay bixisaa sirta dhamaadka-ilaa-dhamaadka, fariimaha libdhaya, waxaana si weyn ugu kalsoonaadaan xirfadlayaasha amniga.

## Maxay Signal?

| Astaamaha | Sababta ay Muhiim u Tahay |
|-----------|--------------------------|
| Sirta dhamaadka-ilaa-dhamaadka | Oo keliya diraha iyo qaataha ayaa akhriyi kara fariimaha |
| Il furan | Koodhka waa la hubsan karaa amniga |
| Fariimaha libdhaya | Waxay yaraynaysaa xogta haddii qalabka la qaato |
| Aan la kaydin xogta fariimaha | Signal ma kaydiyo diiwaanno yaa aad u dirtay fariin |
| Amniga shaashadda | Waxay ka hortagi kartaa sawir qaadista |
| Xirnaanta diiwaangelinta | Waxay ka hortagtaa weerarrada beddelka SIM |

## Dejinta Shakhsiga

Xubin kasta waa inuu si sax ah u dejiyo Signal:

### Tallaabada 1: Ku Rakib Signal

Ka soo degso oo keliya ilaha rasmiga ah:
- [signal.org](https://signal.org/download/)
- Dukaammada app-ka rasmiga ah (iOS App Store, Google Play)

Weligaa ha ka soo degin goobaha saddexaad.

### Tallaabada 2: Isdiiwaangeli

- Isticmaal lambarka taleefanka aad xakameysato
- Tixgeli isticmaalka lambar aan ku xirnayn aqoonsigaaga (lacag hore, Google Voice)

### Tallaabada 3: Deji Dejinta Amniga

Fur Dejinta Signal oo deji:

**Dejinta Qarsoodi:**
- [x] Xirnaanta Shaashadda: SHID (PIN ama biometric)
- [x] Amniga Shaashadda: SHID (ka hortagtaa sawir qaadista)
- [x] Xirnaanta Diiwaangelinta: SHID (ka hortagtaa afduubka beddelka SIM)
- [x] Had iyo jeer Gudbinta Wicitaannada: Tixgeli SHID (qariyaa cinwaanka IP)

**Dejinta Wadahadalka:**
- [x] Fariimaha Libdhaya ee Caadiga ah: 1 toddobaad (ama ka gaaban kooxaha xasaasiga ah)

### Tallaabada 4: Xaqiiji Lambarada Amniga

Markaad la xiriirto xubnaha kale ee unugga, xaqiiji lambarada amniga:
1. Fur wadahadalka
2. Taabo magaca xiriirka ee sare
3. Taabo "Arag Lambarka Amniga"
4. Isbarbar dhig lambarada qof ahaaneed ama wicitaan muuqaal ah

Tani waxay xaqiijinaysaa inaad la hadlayso qofka saxda ah, ma aha qof isdhigaya.

## Qaab-dhismeedka Kooxda

### Kooxaha La Soo Jeedinayo

Abuur kooxo kala duwan qaybta iyo shaqada kasta:

```
Kooxaha Unugga
├── Isku-dubbaridka Maamulka (oo keliya aasaasayaasha + maamulka)
├── Qaybta Waardiyeyaasha
├── Qaybta Gaadiidleyda
├── Qaybta Khayraadka
├── Qaybta Xulafada Ganacsiga
├── Qaybta Ilaaliyeyaasha Sharciga
└── Ogeysiisyada (hal dhinac, unugga oo dhan)
```

### Magacaynta Kooxda

**Ha isticmaalin magacyo cad.** Halkii "Austin Soteria Gaadiidleyda," isticmaal:
- Magacyo kood ah oo dhexdhexaad ah
- Mawduucyo aan xiriir lahayn (naadiyada buugaagta, kooxaha ciyaaraha, kooxaha xiisaha)
- Lambarro ama xarfo

Ku heshii habka magacaynta unugaaga gudihiisa.

### Dejinta Kooxda

Koox kasta:
- **Fariimaha libdhaya**: SHID (1 toddobaad ama ka yar)
- **Xiriirka kooxda**: DAMI (ka hortagtaa ku biirista iyagoo aan ogolaansho admin)
- **Ku dar xubnaha**: Admin oo keliya
- **Wax ka bedel xogta kooxda**: Admin oo keliya

### Dhaqamada Admin

- Ugu yaraan 2 admin koox kasta (kaabayaal dheeraad ah)
- Si degdeg ah u saar xubnaha marka ay baxaan
- Si joogto ah u dib u eeg liiska xubnaha
- Qalabka admin waa inuu leeyahay amniga ugu sarreeya

## Borotokoolada Isgaarsiinta

### Maxaa Laga Hadlayo Signal

- Isku-dubbaridka iyo qorshaynta
- Jadwalka
- Cusboonaysiinta guud
- Xogta tababarka

### Maxaa LAGA HADLI LA'YAHAY Signal

Xitaa sirta, ka taxaddar:
- Goobaha gaarka ah ee dadka nugul
- Xogta shakhsiga ee faahfaahsan
- Wax kasta oo waxyeello noqon kara haddii ay akhristaan maamulka
- Magacyada xubnaha bulshada ee aan diiwaangashanayn

**U qaado in fariimaha laga yaabo in la jabsado.** Sirtu waxay ilaalinaysaa gudbinta — ma aha qalabka la qabtay.

### Erayada Koodhka (Ikhtiyaari)

Unugyada qaarkood waxay horumariyaan erayada koodhka xaaladaha caadiga ah:
- "Cimiladu way isbeddelaysaa" = dhaqdhaqaaqa fulinta la arkay
- "Naadiga buugaagta waa la joojiyay" = joogso, walaac amni
- "Waxaan u baahanahay cunto karinta" = codsanaya gaadiid

U hay nidaamyada koodhka fudud. Koodhyada adag waa la ilaawaa.
