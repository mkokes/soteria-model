# Manual de Seguridad

La seguridad no es opcional — es cómo nos protegemos mutuamente. Este manual cubre las prácticas esenciales que cada participante debe seguir.

## Por Qué Importa la Seguridad

Las personas que protegemos enfrentan consecuencias reales si son expuestas:
- Detención y deportación
- Separación familiar
- Pérdida de empleo
- Acoso y violencia

Nuestras prácticas de seguridad existen para prevenir estos resultados. Cuando tomamos la seguridad en serio, protegemos a los miembros más vulnerables de nuestra comunidad.

## Principios Básicos de Seguridad

### 1. Asume que el Compromiso es Posible

Ningún sistema es perfectamente seguro. Diseñamos nuestra estructura para que cuando (no si) algo sale mal, el daño esté contenido.

### 2. Minimiza lo que Sabes

No puedes revelar lo que no sabes. Por eso compartimentamos.

### 3. Minimiza lo que Almacenas

Los datos que no existen no pueden ser incautados. Usamos mensajes que desaparecen y evitamos registros.

### 4. Practica Consistentemente

La seguridad solo funciona si todos la practican, siempre. Un desliz puede comprometer a otros.

```

├─────────────────────────────────────────────────────────┤

│                    DISPOSITIVO                          │
│      Bloqueos de teléfono, cifrado, apps seguras        │
├─────────────────────────────────────────────────────────┤
│                    FÍSICA                               │
│      Ubicaciones de reunión, registros en papel         │
├─────────────────────────────────────────────────────────┤
│                    CONDUCTUAL                           │
│      Lo que dices, lo que compartes, en quién confías   │
└─────────────────────────────────────────────────────────┘
```

Cada capa refuerza a las demás. La debilidad en una puede ser compensada por la fortaleza en otras — pero todas importan.

## Requisitos Mínimos para Todos los Participantes

Cada persona en la red debe:

- [ ] Usar Signal para toda comunicación celular
- [ ] Configurar mensajes que desaparecen
- [ ] Bloquear su teléfono con PIN/biométrico
- [ ] Nunca discutir actividades celulares en otras plataformas
- [ ] Nunca compartir información entre segmentos
- [ ] Completar entrenamiento de seguridad antes de participar

Estos no son sugerencias. Son requisitos.

## Referencia Rápida: Hacer y No Hacer

### Hacer

✓ Usar Signal para toda comunicación celular  
✓ Bloquear tu teléfono  
✓ Verificar números de seguridad con contactos importantes  
✓ Usar mensajes que desaparecen  
✓ Confiar en tus instintos sobre comportamiento sospechoso  
✓ Hacer preguntas si algo parece extraño  

### No Hacer

✗ Discutir actividades celulares en redes sociales  
✗ Compartir información entre segmentos  
✗ Mantener registros de actividades o miembros  
✗ Usar correo electrónico o SMS para comunicación sensible  
✗ Tomar fotos en actividades sin consentimiento explícito  
✗ Asumir que la seguridad no se aplica a ti
