# إعداد Signal

Signal هو العمود الفقري للاتصالات لخليتك. إعداده بشكل صحيح أمر حاسم.

## لماذا Signal؟

| الميزة | لماذا تهم |
|--------|-----------|
| **التشفير من طرف إلى طرف** | فقط المرسل والمستقبل يمكنهم قراءة الرسائل |
| **الرسائل المختفية** | المحتوى يحذف تلقائياً |
| **الحد الأدنى من البيانات الوصفية** | Signal يخزن القليل جداً عنك |
| **مفتوح المصدر** | يمكن تدقيق الكود للأمان |
| **مجاني** | لا تكلفة، لا إعلانات، لا بيع بيانات |

## تثبيت Signal

### للجميع

1. قم بتنزيل Signal:
   - **iPhone**: App Store
   - **Android**: Google Play Store
   - **Desktop**: signal.org/download

2. سجل برقم هاتفك
3. اختر اسم ملف شخصي (استخدم اسماً أولاً فقط أو اسماً مستعاراً)
4. تخطى استيراد جهات الاتصال (افعل هذا يدوياً)

## الإعدادات المطلوبة

### الخصوصية والأمان

افتح Signal → الإعدادات → الخصوصية:

- [x] **قفل الشاشة**: تطلب PIN/بيومتري لفتح Signal
- [x] **أمان الشاشة**: منع لقطات الشاشة
- [x] **قفل التسجيل**: منع هجمات تبديل SIM
- [ ] **إيصالات القراءة**: فكر في إيقافها (يقلل البيانات الوصفية)
- [ ] **مؤشرات الكتابة**: فكر في إيقافها

### الرسائل المختفية

اضبط افتراضياً لجميع المحادثات:
- Signal → الإعدادات → الخصوصية → الرسائل المختفية الافتراضية → **أسبوع واحد**

للمحادثات الحساسة، استخدم يوم واحد أو ساعة واحدة.

### الإشعارات

قلل ما يظهر على شاشة القفل:
- Signal → الإعدادات → الإشعارات → إظهار → **الاسم فقط** (أو **لا اسم أو رسالة**)

هذا يمنع المحتوى الحساس من الظهور على شاشة القفل.

## إعداد المجموعات

### أنواع المجموعات

ستحتاج خليتك:

| نوع المجموعة | الغرض | الأعضاء |
|-------------|--------|---------|
| **الوكلاء** | تنسيق على مستوى الخلية | الفريق المؤسس/الوكلاء فقط |
| **مجموعات الأقسام** | تنسيق القسم | قائد القسم + الأعضاء |
| **الإعلانات** | اتصالات على مستوى الخلية | الجميع (للقراءة فقط) |

### إنشاء مجموعة

1. Signal → أيقونة القلم → مجموعة جديدة
2. أضف الأعضاء (فقط الأشخاص الذين أكملوا الانضمام)
3. اسم المجموعة (استخدم أسماء عامة: "مجموعة المجتمع"، ليس "خلية سوتيريا")
4. اضبط صورة المجموعة (اختياري، ابقها عامة)

### إعدادات المجموعة

لكل مجموعة:

1. افتح المجموعة → اسم المجموعة → إعدادات المجموعة
2. **الرسائل المختفية**: اضبط المؤقت (أسبوع واحد للتنسيق اليومي، أقصر للحساس)
3. **الأذونات**:
   - من يمكنه تحرير معلومات المجموعة: **المسؤولون فقط**
   - من يمكنه إرسال رسائل: **جميع الأعضاء** (أو **المسؤولون فقط** للإعلانات)
   - من يمكنه إضافة أعضاء جدد: **المسؤولون فقط**

### المسؤولون

اجعل قادة الأقسام/الوكلاء مسؤولين:
1. افتح المجموعة → اسم المجموعة → عرض الأعضاء
2. اضغط على العضو → اجعله مسؤولاً

## التحقق من أرقام الأمان

أرقام الأمان تؤكد أنك تتحدث إلى الشخص الحقيقي.

### متى تتحقق

تحقق من أرقام الأمان مع:
- جميع الأعضاء المؤسسين (مطلوب)
- قادة الأقسام (مطلوب)
- أي شخص تتواصل معه بانتظام (موصى به)

### كيفية التحقق

**شخصياً (الأفضل):**
1. افتح المحادثة → اسم جهة الاتصال → عرض رقم الأمان
2. قارن الأرقام أو امسح رمز QR
3. اضغط "علامة كمتحقق منه"

**عبر مكالمة فيديو (جيد):**
1. ابدأ مكالمة فيديو Signal
2. تحقق من أنك تتحدث إلى الشخص الصحيح
3. ثم قارن أرقام الأمان

::: warning لا تتحقق عبر الرسائل النصية
لا ترسل أرقام الأمان عبر الرسائل النصية أو البريد الإلكتروني. هذا يهزم الغرض.
:::

## أفضل الممارسات

### تسمية جهات الاتصال

استخدم أسماء أولى أو أسماء مستعارة، ليس أسماء كاملة:
- جيد: "ماريا"، "أليكس"
- تجنب: "ماريا غارسيا"، "أليكس تشين"

### نظافة المجموعة

- **أزل الأعضاء فوراً** عندما يغادرون
- **راجع العضوية بانتظام** — هل يجب أن يكون الجميع هنا؟
- **لا تضف أعضاء بدون فحص مناسب**

### الاتصالات

- **استخدم Signal فقط** للاتصالات الخلوية
- **لا رسائل نصية، لا WhatsApp، لا بريد إلكتروني** للمحتوى الحساس
- **لا وسائل تواصل اجتماعي** لمناقشة أنشطة الخلية

### ما لا تشاركه

حتى على Signal، تجنب:
- أسماء محددة لأعضاء المجتمع الذين يتلقون المساعدة
- عناوين أو مواقع مفصلة
- صور للأنشطة أو الأشخاص
- معلومات عن أقسام أخرى (التقسيم)

## استكشاف الأخطاء

### "رقم الأمان تغير"

هذا يحدث عندما:
- شخص ما أعاد تثبيت Signal
- شخص ما حصل على هاتف جديد
- شخص ما غير رقمه

**ماذا تفعل:**
1. تحقق مع الشخص (عبر قناة منفصلة أو شخصياً)
2. إذا كان متوقعاً، علامة كمتحقق منه مرة أخرى
3. إذا كان غير متوقع، كن حذراً — قد يكون الجهاز مخترقاً

### الرسائل لا تختفي

تحقق من:
- هل الرسائل المختفية مفعلة لهذه المحادثة؟
- هل المؤقت اضبط بشكل صحيح؟
- الرسائل تختفي بعد قراءتها + المؤقت، ليس فوراً

### الإشعارات لا تعمل

- تحقق من إعدادات الهاتف → الإشعارات → Signal
- تأكد من أن Signal غير في وضع "عدم الإزعاج"
