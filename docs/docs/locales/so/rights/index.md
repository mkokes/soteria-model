# Ogow Xuquuqdaada

Waxaad leedahay xuquuqo iyadoo aan loo eegin xaaladaada socdaalka. Ogaanshahoodu waxay ilaali karaan adiga iyo qoyskaaga.

::: warning TANI MA AHA TALO SHARCI
Macluumaadkani waa hagid guud. Talo sharci oo ku gaar ah xaaladaada, la tasho qareen socdaal.
:::

## Xuquuqdaada Aasaasiga

### 1. Xaqa Inaad Aammusto

Ma aha inaad ka jawabto su'aalaha ku saabsan meesha aad ku dhalatay, xaaladaada socdaalka, ama sida aad dalka ugu soo gashay.

**Dheh**: "Waxaan isticmaalayaa xaqayga inaan aammuuso."

### 2. Xaqa Inaad Diido Baaritaan

Waxaad xaq u leedahay inaad diido ogolaanshaha baaritaan adiga, baabulkaaga, ama gurigaaga.

**Dheh**: "Oggolaansho uma bixinayo baaritaan."

### 3. Xaqa Inaad Hesho Qareen

Haddii lagu xidho, waxaad xaq u leedahay qareen.

**Dheh**: "Waxaan rabaa inaan la hadlo qareen."

### 4. Xaqa Inaad Waxba Saxiixin

Ha saxiixin wax dukumeenti ah adigoo aan la hadlin qareen marka hore.

### 5. Xaqa Wicitaan Telefoon

Haddii lagu xidho, waxaad xaq u leedahay wicitaan waqti macquul ah gudihiis.

## U Diyaarso Qoyskaaga

1. **Xasuuso lambar telefoon degdeg** — Laga yaabo inaadan heli karin telefoonkaaga
2. **Samee qorshe carruurta** — Ogow cida daryeeli doonta haddii aadan joogin
3. **Qaado kaarka xuquuqda** — Ku hayso xuquuqdaada qoraal ahaan
4. **Ogow qareenkaaga** — Macluumaadka la xiriirka qareen socdaal diyaar u hayso
5. **Qor macluumaad muhiim ah** — Lambarka qariibka (A-number), macluumaadka xiriirka
