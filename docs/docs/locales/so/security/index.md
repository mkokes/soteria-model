# Buugga Amniga

Amniga maaha mid ikhtiyaari ah — waa sida aan isugu ilaalino. Buuggan wuxuu daboolayaa dhaqannada muhiimka ah ee qof kasta oo ka qayb qaata uu raacayo.

## Maxay Amniga Muhiim u Tahay

Dadka aan ilaalino waxay la kulmaan cawaaqib dhabta ah haddii la soo bandhigo:
- Xabsi iyo masaafurin
- Kala soocida qoyska
- Luminta shaqada
- Cadaadis iyo rabshad

Dhaqannadeenna amniga waxay jiraan si ay u hortagaan natiijooyinkan. Marka aan amniga si dhab ah u qaadanno, waxaan ilaalinaa xubnaha ugu nugul ee bulshadeenna.

## Mabaadi'da Aasaasiga ah ee Amniga

### 1. U Qaado in Khatar Suurtogal Tahay

Nidaam kasta ma aha mid si buuxda u badbaad ah. Waxaan u qaabeynaa qaab dhismeedkeenna si marka (maaha haddii) wax khaldan dhacaan, waxyeeladu ay xaddidan tahay.

### 2. Yareey Waxaad Ogtahay

Ma aad shaacin karin waxaad aan garanayn. Tani waa sababta aan u qaybino.

### 3. Yareey Waxaad Kaydisay

Xogta aan jirin lama qaadan karo. Waxaan isticmaalnaa farriimo libdhaya oo aan ka fogaanno diiwaannada.

### 4. Ku Celceli si Joogto ah

Amniga kaliya wuu shaqeeyaa haddii qof walba sameeyo, mar walba. Khalad keliya ayaa khatar gelin kara kuwa kale.

## Lakabka Amniga

```
┌─────────────────────────────────────────────────────────┐
│                    QAAB DHISMEED                        │
│         Qaybinta, taageero, unugyo                      │
├─────────────────────────────────────────────────────────┤
│                    ISGAARSIINTA                         │
│         Farriimaha sirta ah, farriimaha libdhaya        │
├─────────────────────────────────────────────────────────┤
│                    QALABKA                              │
│         Xidhmada taleefanka, sirta, apps badbaadsan     │
├─────────────────────────────────────────────────────────┤
│                    JIREED                               │
│         Goobaha kulanka, diiwaannada warqadda           │
├─────────────────────────────────────────────────────────┤
│                    DHAQAN                               │
│         Waxaad sheegtid, waxaad wadaagtid, cidaad aaminto│
└─────────────────────────────────────────────────────────┘
```

Lakab kasta ayaa xoojinaya kuwa kale. Daciifnimada mid waxaa lagu magdhabi karaa xoogga kuwa kale — laakiin dhammaantood way muhiim yihiin.

## Shuruudaha Ugu Yar ee Dhammaan Ka-qaybgalayaasha

Qof kasta oo shabakada ku jira waa inuu:

- [ ] Isticmaalo Signal dhammaan isgaarsiinta unugga
- [ ] Habeey farriimaha libdhaya
- [ ] Xidho telefoonkooda PIN/bayoomeetariga
- [ ] Waligaa ha ka hadlin hawlaha unugga goobaha kale
- [ ] Waligaa ha wadaagin macluumaadka qaybaha kala duwan
- [ ] Dhammaystir tababarka amniga ka hor ka-qaybgalka

Kuwani maaha talo. Waa shuruudo.

## Tixraac Degdeg ah: Samee iyo Ha Samayn

### Samee

✓ Isticmaal Signal dhammaan isgaarsiinta unugga  
✓ Xidho telefoonkaaga  
✓ Xaqiiji lambarrada amniga xiriiriyaasha muhiimka ah  
✓ Isticmaal farriimaha libdhaya  
✓ Ku kalsoonow dareenkaaga dhaqanka shakiga leh  
✓ Weydiiso su'aalaha haddii wax yaab leh u muuqdaan  

### Ha Samayin

✗ Ka hadal hawlaha unugga baraha bulshada  
✗ Wadaag macluumaadka qaybaha kala duwan  
✗ Hay diiwaannada hawlaha ama xubnaha  
✗ Isticmaal email ama SMS isgaarsiinta xasaasiga ah  
✗ Qaado sawirro hawlaha aan ogolaansho cad lahayn  
✗ U qaado in amniga kugu khusayn
