# Kordhinta

Qaabka Soteria waxaa loo qaabeeyey inuu kordho — labadaba unug gudihiisa iyo bulshooyinka oo dhan. Kordhinta had iyo jeer waa **ikhtiyaari** oo **si maxalli ah loo go'aamiyey**.

## Kordhinta Unug Gudihiisa

Marka unug koro, waxay u baahan kartaa inay qaybiso culayska shaqada:

### Ku Darista Maamule-wadaag

Marka hal maamule la daalayo:

| Calaamadda | Xalka |
|------------|-------|
| Waqtiyada jawaabta oo gaabinaya | Ku dar maamule-wadaag si loo wadaago culayska |
| Hal meel oo fashilid | Tababar maamule kaabto |
| Halista daalka | Wareejinta mas'uuliyadaha |

### Kala-qaybinta Qaybaha

Marka qayb ka korto ~15 xubin, ka fiirso:

- **Kala-qaybinta juquraafiga**: Bari/Galbeed, Waqooyi/Koonfur, xaafad kasta
- **Kala-qaybinta shaqada**: Darawallada vs. direyaasha, kala-soocayaasha vs. qaybiyeyaasha
- **Kala-qaybinta waqtiga**: Kooxda maalinta vs. kooxda fiidka

Qayb-hoose kasta waxay heshaa kooxdeeda Signal iyo hogaamiye.

## Goorta AANAN La Kordhin

Kordhinta waa ikhtiyaari. Xaaladaha qaar waxay si fiican ugu shaqeeyaan iyagoo yar:

| Xaaladda | Talooyinka |
|----------|------------|
| Tuulo yar/fog | Unug keliya ayaa u adeegi kara si aan xad lahayn |
| Heer dhaqdhaqaaq hooseeya | Ha abuurin qaab-dhismeed aadan u baahnayn |
| Shabakadda kalsoonida weli horumaraysa | Kor u qaad kalsoonida ka hor qaab-dhismeedka |
| Cadaadiska dibadda oo kordhaya | Isku-ururi halkii laga balaarin lahaa |

**Yar waxay noqon kartaa ammaan.** Qaab-dhismeed badan waxay ka dhigan tahay adag badan iyo meelo badan oo fashilid suurtagal ah.

## Dhalinta Unugyo Cusub

Marka bulsho cusub rabto inay bilowdo:

### Habka Kooxda Abuurka

1. **Aqoonsi aasaasayaasha**: 3-5 qof aagga cusub, ay dammaanayaan shabakadda jirta
2. **Tababar**: Aasaasayaasha waxay helaan tababar buuxa heerka maamulaha
3. **Madax-bannaanida**: Unugga cusub wuxuu u shaqeeyaa si madax-bannaan maalinta koowaad
4. **Kala-sareyn la'aan**: Unugga asalka ah ma laha awood unugga cusub

### Wadaagista Aqoonta (Ikhtiyaari)

Unugyadu waxay dooran karaan inay wadaagaan:
- Macluumaadka sharciga ee la cusboonaysiiyey
- Hagaajinta dhaqamada amniga
- Hagaajinta qaababka
- Casharrada guud ee la bartay

Unugyadu WAAJIBAAD MA AHA inay wadaagaan:
- Liisaska xubnaha
- Faahfaahinta hawlgalka
- Kanaalada xiriirka
- Faahfaahinta qaab-dhismeedka gudaha

### Qaabka Xiriiriyaha (Ikhtiyaari)

Haddii unugyadu rabaan xiriir joogto ah:

```
┌──────────────┐          ┌──────────────┐
│   Unug A     │          │   Unug B     │
│              │          │              │
│  ┌────────┐  │          │  ┌────────┐  │
│  │Xiriiriye│◄─┼──────────┼─►│Xiriiriye│  │
│  └────────┘  │          │  └────────┘  │
└──────────────┘          └──────────────┘
```

- Hal qof oo la magacaabay unug kasta xiriirka unugyada dhexdooda
- Wuxuu wadaagaa kaliya macluumaadka aan hawlgalka ahayn
- Ma u ballan qaadi karo unuggiisa waxba

## Mabaadi'da Kordhinta

1. **Kordhi marka loo baahdo, ma aha marka suurtagal tahay** — Ha abuurin qaab-dhismeed ujeedadiisa
2. **Go'aannada maxalliga ah** — Unug/qayb kasta ayaa go'aamiya goorta iyo sida loo kordho
3. **Ilaali kala-soocidda** — Kordhinta waa inaysan jebin xadka amniga
4. **Diiwaangeli markaad koraysid** — Qaab-dhismeedyada waaweyn waxay u baahan yihiin habab cad
5. **Madax-bannaanida ka sarraysa waxqabadka** — Unugyada madax-bannaaniga ah way ka ammaan badan yihiin kuwa isku-dubarisan
