## قائمة تحقق الانضمام

```markdown
# قائمة تحقق انضمام العضو الجديد

**اسم المرشح:** _______________
**المزكي:** _______________
**التاريخ المقترح:** _______________

## المرحلة 1: التزكية
- [ ] المزكي يعرف المرشح لمدة 6+ أشهر
- [ ] المزكي يثق بالمرشح بمعلومات حساسة
- [ ] المزكي يشهد على قيم المرشح والتزامه
- [ ] المزكي يفهم المسؤولية

## المرحلة 2: الاجتماع الأولي
- [ ] المزكي شرح غرض الخلية
- [ ] المزكي شرح توقعات الالتزام
- [ ] المزكي شرح متطلبات الأمان
- [ ] المرشح أعرب عن اهتمامه بالمتابعة

## المرحلة 3: موافقة الوكلاء
- [ ] المزكي قدم المرشح للوكلاء
- [ ] الوكلاء راجعوا التزكية
- [ ] الوكلاء وافقوا على المتابعة
- [ ] **التاريخ:** _______________

## المرحلة 4: التدريب الأمني
- [ ] المرشح قرأ دليل الأمان
- [ ] المرشح ثبت وكون Signal
- [ ] المرشح حضر جلسة تدريب أمان
- [ ] المدرب أكد الفهم
- [ ] **المدرب:** _______________ **التاريخ:** _______________

## المرحلة 5: تعيين القسم
- [ ] القسم المعين: _______________
- [ ] قائد القسم أُبلغ
- [ ] قائد القسم وافق

## المرحلة 6: الإضافة إلى Signal
- [ ] أُضيف إلى مجموعة القسم
- [ ] تحقق من رقم الأمان مع قائد القسم
- [ ] تحقق من رقم الأمان مع المزكي
- [ ] أُضيف إلى مجموعة الإعلانات (إذا كانت موجودة)

## المرحلة 7: الدمج
- [ ] المزكي تحقق بعد أسبوع واحد
- [ ] قائد القسم تحقق بعد أسبوع واحد
- [ ] مراجعة شهر واحد مكتملة
- [ ] **حالة:** _______________
```
