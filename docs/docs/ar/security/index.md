# دليل الأمان

الأمان ليس اختياريًا — إنه كيف نحمي بعضنا البعض. يغطي هذا الدليل الممارسات الأساسية التي يجب على كل مشارك اتباعها.

## لماذا يهم الأمان

الأشخاص الذين نحميهم يواجهون عواقب حقيقية إذا تم كشفهم:
- الاحتجاز والترحيل
- انفصال الأسرة
- فقدان الوظيفة
- المضايقة والعنف

ممارساتنا الأمنية موجودة لمنع هذه النتائج. عندما نأخذ الأمان على محمل الجد، نحمي الأعضاء الأكثر ضعفًا في مجتمعنا.

## المبادئ الأساسية للأمان

### 1. افترض أن الاختراق ممكن

لا يوجد نظام آمن تمامًا. نصمم هيكلنا بحيث عندما (وليس إذا) يحدث خطأ ما، يكون الضرر محتويًا.

### 2. قلل ما تعرفه

لا يمكنك الكشف عما لا تعرفه. لهذا السبب نقوم بالتقسيم.

### 3. قلل ما تخزنه

البيانات التي لا توجد لا يمكن الاستيلاء عليها. نستخدم الرسائل المختفية ونتجنب السجلات.

### 4. مارس باستمرار

الأمان يعمل فقط إذا مارسه الجميع، دائمًا. زلة واحدة يمكن أن تعرض الآخرين للخطر.

## طبقات الأمان

```
┌─────────────────────────────────────────────────────────┐
│                    هيكلي                                │
│      التقسيم، الضمان، الخلايا                          │
├─────────────────────────────────────────────────────────┤
│                    الاتصالات                            │
│      المراسلة المشفرة، الرسائل المختفية                │
├─────────────────────────────────────────────────────────┤
│                    الجهاز                               │
│      أقفال الهاتف، التشفير، التطبيقات الآمنة           │
├─────────────────────────────────────────────────────────┤
│                    مادي                                 │
│      مواقع الاجتماعات، السجلات الورقية                 │
├─────────────────────────────────────────────────────────┤
│                    سلوكي                                │
│      ما تقوله، ما تشاركه، من تثق به                   │
└─────────────────────────────────────────────────────────┘
```

كل طبقة تعزز الأخرى. الضعف في واحدة يمكن تعويضه بالقوة في أخرى — لكن كلها مهمة.

## الحد الأدنى من المتطلبات لجميع المشاركين

يجب على كل شخص في الشبكة:

- [ ] استخدام Signal لجميع اتصالات الخلية
- [ ] إعداد الرسائل المختفية
- [ ] قفل هاتفهم برقم التعريف الشخصي/البيومتري
- [ ] عدم مناقشة أنشطة الخلية على منصات أخرى أبدًا
- [ ] عدم مشاركة المعلومات عبر القطاعات أبدًا
- [ ] إكمال التدريب الأمني قبل المشاركة

هذه ليست اقتراحات. إنها متطلبات.

## مرجع سريع: افعل ولا تفعل

### افعل

✓ استخدم Signal لجميع اتصالات الخلية  
✓ اقفل هاتفك  
✓ تحقق من أرقام الأمان مع جهات الاتصال المهمة  
✓ استخدم الرسائل المختفية  
✓ ثق في غرائزك حول السلوك المشبوه  
✓ اطرح أسئلة إذا بدا شيء غريبًا  

### لا تفعل

✗ ناقش أنشطة الخلية على وسائل التواصل الاجتماعي  
✗ شارك المعلومات عبر القطاعات  
✗ احتفظ بسجلات للأنشطة أو الأعضاء  
✗ استخدم البريد الإلكتروني أو الرسائل النصية للاتصالات الحساسة  
✗ التقط صورًا في الأنشطة بدون موافقة صريحة  
✗ افترض أن الأمان لا ينطبق عليك
